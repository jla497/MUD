Gibanica MUD README
===================

### Packages Needed

You may need to install these packages

- boost
- libyaml-cpp-dev

### Usage

Server:
- ./mudserver <port> <yaml_files(s)>

Client:
- ./chatclient <host_ip> <port>

### Actions Supported
- say 
- move 
- attack (dummy attack)
- look (at the room)
